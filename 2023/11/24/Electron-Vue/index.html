Electron+Vue

Author: Echo Date: November 24, 2023  10:21:40 Category: Electron

This blog chronicled the process of building the electron project. I want to use electron + vue3 + ts + antv to develop a mind mapping project.

  • Firstly, I use vite to build a vue3 project, like this :

yarn create vite ele-vue-antv

  • Then, install the relevant dependencies in this project

After the dependencies have been installed, let’s configure Electron.

  • create a directory named electron, and touch a file named index.ts

  • Configure thr electron entry file: open vite.config.ts, and import electron/index.ts

  • Next, let’s configure the package.json file, add the ‘main’ field, remove the ‘type’ field

  • Finally, try running yarn dev to see the results.

I haven’t successfully packaged it on macOS yet. I will provide further updates in this article
