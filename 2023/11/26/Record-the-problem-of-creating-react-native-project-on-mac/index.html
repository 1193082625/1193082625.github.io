Record the problem of creating react native project on mac

Author: Echo Date: November 26, 2023  15:59:17

I’ve been stuck with a question for one day. It’s about running react native project.

In the beginning, I successfully created a project, and it works normally.

But, when I tried to use Material Top Tabs, I got problems. It cannot run properly.

Then, I tried to create a new project, it doesn’t work either.

When I create new project, it shows error that gem can’t be installed because there is no write permission.

After many attempts, it was solved by doing the following

  • npx react-native@latest init DuoXi

    The first time I run this command I got two errors, gem and cocoapods are installed failed. Then I use sudo to rerun command.

  • sudo npx react-native@latest init DuoXi

    This time, it shows error that the cocoapods is installed failed . But this time, the gem has been successfully installed using sudo.

  • npx react-native@latest init DuoXi2

    Lastly, its normal creation is successful.
